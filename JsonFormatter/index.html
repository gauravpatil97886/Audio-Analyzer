origin story
Why this tool exists
While debugging a production API response, Saurabh Yadav from the Tech Support team flagged a recurring pain point — extracting specific field values across large JSON arrays was tedious and time-consuming. Every time a support issue came in, the team had to manually scan through hundreds of records just to pull out IDs, statuses, or names. Gaurav Patil built this tool to solve exactly that — paste any JSON, click any key, get all values instantly. No setup. No server. Just results.
